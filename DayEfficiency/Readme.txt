Перед первым запуском, необходимо в файле конфигурации изменить следующие элементы:
	"source_file" - полное имя .xlsx файла с исходной таблицей;
	"destination_file" - полное имя .txt файла назначения;
	"cell_name" - адрес ячейки исходной таблици;
	"processing_time" - время доступное для чтения информации из таблици (до 24:00).

В дальнейшем, при внесении изменений в файле конфигурации (например для смены адреса ячейки), 
либо при необходимости начать обработку заново. Необходимо изменить значение элемента
	"last_cell_value" на -1 (минус один).
